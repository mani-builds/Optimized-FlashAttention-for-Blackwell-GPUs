* Phase 1 (Reimplementation of FA-2 from scartch on 4080)
Week 1 and 2
This is where you learn everything. Follow the original FlashAttention-1 paper + Tri Dao’s blog (links below). Goal: write your own flash_attn_forward kernel that beats torch.nn.functional.scaled_dot_product_attention.

Milestones:

** DONE Day 1–3: Read + implement basic tiled matmul + naive softmax (just to feel the pain).
** DONE Day 4–7: Add shared memory tiling + online softmax (the real FlashAttention trick).
** TODO Day 8–10: Warp-level reductions, causal masking, PyTorch custom op integration (torch.utils.cpp_extension).
